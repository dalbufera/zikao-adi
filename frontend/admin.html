Documentation API

POST /api/v1/chat
Discuter avec Zikao. Retourne le texte, l'emotion et la musique si demandee.
curl -X POST https://VOTRE-SERVEUR:3100/api/v1/chat \ -H "Authorization: Bearer zk_votre_cle" \ -H "Content-Type: application/json" \ -d '{"message": "joue du Tupac", "userId": "app1"}'
GET /api/v1/music/search?q=...&type=track|artist
Rechercher de la musique (artistes ou morceaux via Deezer).
curl https://VOTRE-SERVEUR:3100/api/v1/music/search?q=tupac \ -H "Authorization: Bearer zk_votre_cle"
GET /api/v1/status
Etat actuel de Zikao (humeur, activite, disponibilite).
curl https://VOTRE-SERVEUR:3100/api/v1/status \ -H "Authorization: Bearer zk_votre_cle"
